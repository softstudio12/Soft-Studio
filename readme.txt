# Soft Studio — Static Website

Soft Studio is a clean, responsive website built entirely with HTML and CSS.  
It’s designed for small fashion brands or anyone who wants a simple, modern front-end.

## Features
- Pure HTML + CSS (no JavaScript)
- Mobile responsive
- Product cards and clean layout
- Inline SVG logo
- Easy to customize

## How to Run
1. Download or clone the repository  
2. Open the folder  
3. Double-click `index.html`  
4. The website opens instantly  

No server. No terminal. No installation needed.

## Customize
All styling is inside `<style>` in `index.html`.  
You can change colors, fonts, images, or layout freely.
